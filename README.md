### 🤖 [WhatsApp Chat Sentiment Analyzer](https://github.com/RehmanSiddique/WhatsApp-Chat-Sentiment-Analysis)
> NLP pipeline that processes exported WhatsApp chats and surfaces sentiment + behavioral insights

- Parses raw `.txt` chat exports → NLP sentiment classification (positive / negative / neutral) per user over time
- Generates 8 chart types: activity heatmaps, emoji analysis, word clouds, per-user sentiment breakdowns
- Custom `stop_hinglish.txt` corpus for Pakistan/India Urdu–English code-switching — built from scratch
- **Stack:** Python · NLTK · Streamlit · Pandas · Seaborn · WordCloud
